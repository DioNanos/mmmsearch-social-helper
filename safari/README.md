# Safari Wrapper Plan

Status: not implemented from this Linux VPS.

Safari iPhone / iPad / macOS requires:
- Safari Web Extension target
- wrapper app created in Xcode
- Apple signing and distribution via TestFlight or App Store

This repo is the source base, but the actual Safari packaging must be done on macOS with Xcode.

Public rule:
- do not claim Safari installability until the wrapper app exists and is published
