# Chrome Web Store Listing Draft

## Name
mmmsearch social helper

## Short description
Improve Profile Scan on Instagram and Facebook using your local browser session.

## Detailed description
mmmsearch social helper improves social profile extraction for mmmsearch on sites that are difficult to read reliably from a server IP.

The extension runs in your browser and can use your local logged-in session on supported sites like Instagram and Facebook.

What it does:
- extracts profile fields such as name, bio, avatar, counters, and external link
- sends only extracted profile fields back to mmmsearch for the current request
- does not send your cookies or credentials to mmmsearch servers

Current supported runtime targets:
- Instagram
- Facebook

Current supported browsers:
- Chrome
- Brave
- Edge

Notes:
- best results require being logged in on the supported site in your own browser
- Chrome mobile does not run this extension locally on the phone
